# NIT Faculty Web Scrapper

In this project, data about faculty from the websites of various NIT is collected and stored in a docx file.

## Libraries Used

- Selenium
- BeautifulSoup ( bs4 )

## NIT Scrapped Till Now in this project

1. NIT Uttarakhand
2. NIT Puducherry
3. NIT Arunachal Pradesh
